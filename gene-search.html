How to Use Gene Search

Search Features

  • Text Search: Type gene symbol (e.g., "Cacna1g"), gene name (e.g., "calcium channel"), or Ensembl ID
  • Regulation Filter: Show only upregulated or downregulated genes
  • Dataset Filter: Include/exclude specific datasets
  • Statistical Filters: Filter by p-value or fold change thresholds

Table Features

  • Sortable Columns: Click column headers to sort (↑↓)
  • Pagination: Browse results with Previous/Next buttons
  • Details Button: View complete gene information and external links
  • Export: Download filtered results as CSV for further analysis

Dataset Information

  • Bulk RNA-seq: 593 DEGs from LCM-based bulk RNA-seq (Ga30 vs Ga15)
  • Spatial P15: 1,079 DEGs from 10X Visium spatial transcriptomics at P15
  • Spatial P30: 1,079 DEGs from 10X Visium spatial transcriptomics at P30
  • Seizure Genes: 2,007 genes associated with seizure phenotype (HP:0001250)
  • Total Unique Genes: 3,611 genes across all datasets
